Release Notes for FP-SNS-MOTENV1 Projects

Copyright © 2021 STMicroelectronics

Purpose

The FP-SNS-MOTENV1 Projects provide applications to aid understanding how to develop an application using the BLE Middleware for the BlueNRG-2 device and Motion libraries middlewares. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals and supported expansion nucleo boards. STM32CubeMX project files (.ioc) can be used for generate the MOTENV1 and BootLoader application for each supported board.
